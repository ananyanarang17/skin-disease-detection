Technology Behind the App

This application is powered by a deep learning model trained on the HAM10000 dataset using TensorFlow and Keras. The frontend runs on HTML, CSS, and JavaScript, while the backend uses Flask (Python) for integration, image processing with OpenCV, and report generation through FPDF. AI chat assistance is powered via OpenRouter API.

Our Project Journey

Starting as a simple college project, AI SkinCare evolved into a fully functional web app through continuous experimentation and teamwork. From training models to designing an intuitive interface, each phase focused on creating a user-friendly and educational experience.
